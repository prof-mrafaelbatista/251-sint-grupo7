Estruturas de Seleção

Conceito

Imagine que seu programa é um GPS. As estruturas de seleção são como as bifurcações na estrada: elas permitem que o programa escolha qual caminho seguir com base em certas condições. Usamos principalmente if (se), elif (senão se) e else (senão). O programa testa se uma condição é verdadeira ou falsa e, dependendo do resultado, executa um bloco de código específico.

Aplicação

São super úteis para:

  • Verificar se um usuário digitou a senha correta.
  • Decidir se um aluno foi aprovado ou reprovado com base na nota.
  • Mostrar mensagens diferentes dependendo da escolha do usuário em um menu.

Exemplo de Código e Explicação

Código Python

# Exemplo 1: Maioridade
idade = 18
if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

# Exemplo 2: Conceito por nota
nota = 7.5
if nota >= 9.0:
    print("Conceito A: Excelente!")
elif nota >= 7.0:
    print("Conceito B: Bom trabalho!")
else:
    print("Conceito C: Precisa melhorar.")
Explicação Linha a Linha

Exemplo 1: Maioridade

idade = 18
Criamos uma variável idade e guardamos o número 18 nela.
if idade >= 18:
Se a idade for maior ou igual a 18, execute o próximo bloco.
    print("Você é maior de idade.")
Mostra esta mensagem se a condição do if for verdadeira. O espaço no início (indentação) é crucial!
else:
Senão (se a condição do if for falsa)...
    print("Você é menor de idade.")
Mostra esta mensagem.

Exemplo 2: Conceito por nota

nota = 7.5
Guardamos 7.5 na variável nota.
if nota >= 9.0:
Se a nota for maior ou igual a 9.0...
    print("Conceito A: Excelente!")
Mostra "Conceito A".
elif nota >= 7.0:
Senão se (se o if anterior foi falso) a nota for maior ou igual a 7.0...
    print("Conceito B: Bom trabalho!")
Mostra "Conceito B".
else:
Senão (se nenhuma das condições anteriores foi verdadeira)...
    print("Conceito C: Precisa melhorar.")
Mostra "Conceito C".

Estruturas de Repetição

Conceito

São como dar "replay" em uma parte do seu código. Elas permitem que você execute o mesmo bloco de instruções várias vezes. As principais são o for (para cada item em uma coleção, faça algo) e o while (enquanto uma condição for verdadeira, continue fazendo algo).

Aplicação

Muito usadas para:

  • Mostrar todos os nomes de uma lista de contatos.
  • Calcular a soma de vários números.
  • Fazer um jogo continuar até que o jogador decida sair ou perca.

Exemplo de Código e Explicação

Código Python

# Exemplo com for: Listando frutas
frutas = ["maçã", "banana", "laranja"]
for fruta_atual in frutas:
    print(f"Eu gosto de {fruta_atual}")

# Exemplo com while: Contagem regressiva
contador = 5
while contador > 0:
    print(contador)
    contador = contador - 1 # ou contador -= 1
print("Decolar!")
Explicação Linha a Linha

Exemplo com for: Listando frutas

frutas = ["maçã", "banana", "laranja"]
Criamos uma lista chamada frutas.
for fruta_atual in frutas:
Para cada fruta_atual na lista frutas, execute o bloco abaixo. A cada rodada, fruta_atual recebe um item da lista.
    print(f"Eu gosto de {fruta_atual}")
Mostra a mensagem com a fruta da rodada atual.

Exemplo com while: Contagem regressiva

contador = 5
Iniciamos a variável contador com 5.
while contador > 0:
Enquanto o contador for maior que 0, execute o bloco.
    print(contador)
Mostra o valor atual do contador.
    contador = contador - 1
Diminui o contador em 1. Essencial para o loop terminar!
print("Decolar!")
Executado após o loop while terminar.

Vetores e Matrizes (Listas em Python)

Conceito

Vetores (ou listas em Python) são coleções ordenadas de itens, como gavetas numeradas (índices a partir de 0). Matrizes são como tabelas, representadas em Python como "listas de listas".

Aplicação

Listas guardam sequências (notas, nomes). Matrizes representam dados tabulares (tabuleiros, planilhas).

Exemplo de Código e Explicação

Código Python

# Vetor (Lista) de compras
compras = ["pão", "leite", "ovos"]
print(compras[0]) # Acessa "pão"
compras.append("café") # Adiciona "café"

# Matriz (Lista de Listas) - Notas
# [Prova1, Prova2]
notas_turma = [
    [7.5, 8.0],  # Aluno 1
    [9.0, 8.5]   # Aluno 2
]
print(notas_turma[0][1]) # Nota Prova2 Aluno1
Explicação Linha a Linha

Vetor (Lista) de compras

compras = ["pão", "leite", "ovos"]
Cria uma lista compras.
print(compras[0])
Mostra o item no índice 0 (primeiro item: "pão").
compras.append("café")
Adiciona "café" ao final da lista compras.

Matriz (Lista de Listas) - Notas

notas_turma = [ [7.5, 8.0], [9.0, 8.5] ]
notas_turma é uma lista. Cada item dela é outra lista de notas.
[7.5, 8.0] são as notas do Aluno 1 (índice 0 da notas_turma).
[9.0, 8.5] são as notas do Aluno 2 (índice 1 da notas_turma).
print(notas_turma[0][1])
notas_turma[0] pega a lista do Aluno 1: [7.5, 8.0].
Dessa lista, [1] pega o item no índice 1, que é 8.0 (Nota da Prova 2 do Aluno 1).

Funções e Procedimentos

Conceito

Blocos de código nomeados e reutilizáveis (definidos com def). Funções geralmente retornam um valor (return). Procedimentos realizam ações sem necessariamente retornar um valor significativo (em Python, retornam None implicitamente).

Aplicação

Organizam, reutilizam e tornam o código mais legível.

Exemplo de Código e Explicação

Código Python

# Função que retorna um valor
def somar(a, b):
    resultado_soma = a + b
    return resultado_soma

# Procedimento (ação, sem retorno útil)
def saudar(nome):
    print(f"Olá, {nome}!")

# Usando a função
num1 = 5
soma_final = somar(num1, 3)
print(f"Soma: {soma_final}")

# Usando o procedimento
saudar("Ana")
Explicação Linha a Linha

Função somar

def somar(a, b):
Define uma função chamada somar que aceita dois "ingredientes" (parâmetros): a e b.
    resultado_soma = a + b
Calcula a soma de a e b, guarda em resultado_soma.
    return resultado_soma
Devolve o valor de resultado_soma para quem chamou a função.

Procedimento saudar

def saudar(nome):
Define saudar que aceita um parâmetro nome.
    print(f"Olá, {nome}!")
Mostra uma saudação. Não usa return para devolver um valor específico.

Usando as definições

soma_final = somar(num1, 3)
"Chama" a função somar com num1 (que é 5) e 3. O valor retornado (8) é guardado em soma_final.
print(f"Soma: {soma_final}")
Mostra o resultado da soma.
saudar("Ana")
"Chama" o procedimento saudar, que mostrará "Olá, Ana!".

Tratamento de Exceções

Conceito

Mecanismo para lidar com erros que ocorrem durante a execução (exceções), usando try (tentar código), except (capturar erro), else (se não houver erro) e finally (executar sempre).

Aplicação

Cria programas robustos ao lidar com entradas inválidas, falhas de arquivo/rede, etc.

Exemplo de Código e Explicação

Código Python

try:
    num_str = input("Digite um número: ")
    numero = int(num_str)
    resultado = 10 / numero
except ValueError:
    print("Entrada não é um número válido!")
except ZeroDivisionError:
    print("Não se pode dividir por zero!")
except Exception as e:
    print(f"Outro erro: {e}")
else:
    print(f"Resultado: {resultado}")
finally:
    print("Tentativa finalizada.")
Explicação Linha a Linha
try:
Inicia um bloco onde erros podem ser "tentados" e capturados.
    num_str = input(...)
Pede uma entrada ao usuário (vem como texto).
    numero = int(num_str)
Tenta converter o texto para inteiro. Pode dar ValueError se não for um número (ex: "abc").
    resultado = 10 / numero
Tenta dividir 10 pelo número. Pode dar ZeroDivisionError se numero for 0.
except ValueError:
Se um ValueError ocorreu no bloco try, executa este bloco.
    print("Entrada não é...")
Mensagem para erro de valor.
except ZeroDivisionError:
Se um ZeroDivisionError ocorreu, executa este bloco.
    print("Não se pode...")
Mensagem para divisão por zero.
except Exception as e:
Captura qualquer outro tipo de erro não previsto antes. e contém detalhes do erro.
    print(f"Outro erro: {e}")
Informa sobre o erro genérico.
else:
Executado SOMENTE se NENHUM erro ocorreu no bloco try.
    print(f"Resultado: {resultado}")
Mostra o resultado se tudo deu certo.
finally:
Executado SEMPRE, com ou sem erro, após o try, excepts e else.
    print("Tentativa finalizada.")
Mensagem final, útil para limpeza.
